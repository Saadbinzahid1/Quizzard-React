# 🧠 Quizzard – React Quiz Application

Quizzard is an interactive quiz application built with React to practice and demonstrate advanced state management using the **useReducer** hook.

The app simulates a real quiz experience where users answer questions, track their score, and navigate through the quiz while React manages the application state efficiently.

---

## 🌐 Live Preview

[Live Demo](https://quizzardreact.netlify.app)

---

## 🚀 Features

- 📝 Interactive quiz interface
- ⏱️ Question progression system
- 🎯 Real-time score tracking
- ⚡ Efficient state management using `useReducer`
- 📊 Final score display
- 🔄 Restart quiz functionality
- 💡 Clean and responsive UI

---

## 🛠️ Built With

- React (Functional Components)
- useReducer Hook
- JavaScript (ES6+)
- HTML5
- CSS3

---

## 🎯 Project Purpose

The main goal of this project is to strengthen understanding of **React state management patterns**, particularly when dealing with more complex application logic.

Using `useReducer` helps keep state updates structured, predictable, and easier to scale compared to multiple `useState` hooks.
